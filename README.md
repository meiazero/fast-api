# fast-api
> An example of a FastAPI project

:warning: This project is a work in progress.

## Prerequisites

- Python 3.11 or higher
- Poetry

## Installation

```sh
poetry install
```

## Usage

```sh
poetry shell
```

```sh
task run
```

## Run tests

```sh
task pre_test
```

```sh
task test
```

```sh
task post_test
```
